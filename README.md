# 智慧商店   

## 环境准备   
-[微信小程序开发工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/stable.html)   
node环境

## 安装依赖
```
cd miniprogram && npm install 
**如果npm安装很慢,请执行以下设置**  
npm config set registry https://registry.npm.taobao.org
```
